# tarotmui-legal

塔羅妹 TarotMui 私隱政策與免責聲明（GitHub Pages）。

## 公開網址

部署完成後：

- 私隱政策：`https://YOUR_GITHUB_USERNAME.github.io/tarotmui-legal/privacy.html`
- 免責聲明：`https://YOUR_GITHUB_USERNAME.github.io/tarotmui-legal/disclaimer.html`

## 首次部署

1. 在 GitHub 新建 **Public** repository，名稱：`tarotmui-legal`
2. 將本資料夾內所有檔案 push 到 `main` 分支
3. GitHub → **Settings** → **Pages**
4. **Source**：Deploy from a branch
5. **Branch**：`main` → **`/ (root)`** → Save
6. 等 1–2 分鐘，打開上面的 `privacy.html` 網址確認中文正常

## 更新內容

改完 `privacy.html` 或 `disclaimer.html` 後 push 到 GitHub 即可，無需額外步驟。
